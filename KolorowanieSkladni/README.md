# SkanerTKiK
Język implementacji: Java

Tokeny:

| Kod      | Opis                                |
|----------|-------------------------------------|
| NUM      | Ciąg cyfr                           |
| STR      | Ciąg znaków w cudzysłowie           |
| BOOL     | Prawda lub fałsz                    |
| ID       | Identyfikator zmiennej, ciąg znaków |
| PLUS     | Znak dodawania                      |
| MINUS    | Znak odejmowania                    |
| MULT     | Znak mnożenia                       |
| DIV      | Znak dzielenia                      |
| LBRACKET | Lewy nawias                         |
| RBRACKET | Prawy nawias                        |
| ASSIGN   | Operator przypisania                |
| VAR      | Oznaczenie zmiennej                 |
| EQUAL    | Operator równości                   |
| LESS     | Operator mniejszości                |
| GREATER  | Operator większości                 |
| ELESS    | Operator "mniejsze równe"           |
| EGREATER | Operator "większe równe"            |
| NEQUAL   | Operator różności                   |
| FOR      | Pętla                               |
| IF       | Sprawdzenie logiczne                |
| ELSE     | Alternatyczne sprawdzenie logiczne  |
| INC      | Inkrementacja                       |
| PRINT    | Funkcja wypisania na terminal       |
| ERR      | Nieznany token                      |
| EOF      | Koniec ciągu tokenów                |
