# SkanerTKiK
Język implementacji: Java

Tokeny:

| Kod      | Opis                       |
|----------|----------------------------|
| NUM      | Ciąg cyfr                  |
| ID| Identyfikator, ciąg znaków |
| PLUS     | Znak dodawania             |
| MINUS    | Znak odejmowania           |
| MULT     | Znak mnożenia              |
| DIV      | Znak dzielenia             |
| LBRACKET | Lewy nawias                |
| RBRACKET | Prawy nawias               |
|ERR| Nieznany token             |
|EOF| Koniec ciągu tokenów       |
